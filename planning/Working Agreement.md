# Working Agreement

**Repository:** [github.com/victorbuenog/AstraNotes](https://github.com/victorbuenog/AstraNotes).

## Commitments

- Based on project requirements and my vision, I will keep a feature list updated frequently with status **To-Do**, **In Progress**, or **Done**, so the project stays on track and the vision does not drift.
- I will use AI to plan features, explain pros and cons of different approaches, for sheer code writing, and to document what was done in a comprehensive **log** (prompts and what was implemented each time) and a **README** (summary of features and project structure).
- I will add my own feedback to the log to steer AI agents in future revisions.
- AI output will be tested with **automated tests** and by the **test user (me)**. No new features will be developed until everything from the previous prompt is completely verified.
- While documentation and one-by-one review help limit drift and duplication, occasionally a **fresh** agent will be tasked with a thorough codebase review so legacy features that are no longer needed can be trimmed out.
